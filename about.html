Food Service Establishment Inspection Results for CNY

I built this app to demonstrate how to use open data from government to build a useful application. I'd also like to avoid food poisining if I can.

All of the code for this app is open source, you can find it on GitHub.

If you'd like to help build more apps like this using open data, check out the Upstate Data Project.
